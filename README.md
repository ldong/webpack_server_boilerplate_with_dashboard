# Webpack Server Boilerplate with Dashboard

Author: Lin Dong

Date: March 14, 2016

## Instruction

```
npm install
npm install webpack-dev-server@1.12.1 -g
webpack-dev-server --progress --colors
```

Now open http://localhost:8080/webpack-dev-server/
